## About

### Overview

This project is a backend-only Inventory Management System built using Node.js and Express. It provides a set of RESTful API endpoints to manage inventory items, including creating, updating, deleting, and retrieving data. The system uses an in-memory array for data storage and focuses on clean structure with controllers, routes, and middleware separation.

### Features

- **Inventory CRUD Operations** <br />
  Create, update, delete, and retrieve inventory items through API endpoints.

- **Fetch Single & All Items** <br />
  Supports retrieving all items or a specific item using its ID.

- **Search Functionality** <br />
  Allows searching items by name using query parameters.

- **Filtering** <br />
  Filter items by category (case-insensitive) and quantity.

- **In-Memory Data Storage** <br />
  Uses arrays to store data during runtime without any database.

- **Middleware Support** <br />
  Includes validation middleware for request checking and centralized error handling.

- **RESTful API Design** <br />
  Structured endpoints following REST principles with consistent JSON responses.
